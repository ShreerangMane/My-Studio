IMPORTANT: ENVIRONMENT VARIABLE REQUIRED
=========================================

You MUST add this environment variable in Vercel Dashboard:

Name:  ANTHROPIC_API_KEY
Value: Your API key from console.anthropic.com (starts with sk-ant-...)

Steps:
1. Go to Vercel Dashboard → Your Project
2. Settings → Environment Variables
3. Add New
4. Name: ANTHROPIC_API_KEY
5. Value: [paste your key]
6. Check all three environments
7. Save
8. Go to Deployments → Redeploy

Without this, your backend will NOT work!
